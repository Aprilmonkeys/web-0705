 * Web应用的完整架构
   * 客户端 - 用于与用户进行交互
     * 将数据展示给用户
     * 接收用户输入
   * 服务器端 - 完成业务逻辑
   * 数据库 - 用于存储数据
 * HTML的内容
   * 概念
     * 全称 - 超文本标记语言
       * 超文本 - 在普通的文本基础上，添加超链接、图片、音频或视频等
       * 标记 - 标记就是HTML中的标签(元素)，<a>
     * HTML基本结构
       <!DOCTYPE html> - 声明，定义当前HTML的版本
       <html> - 根标签
       <head> - 定义当前HTML的信息
            <meta charset="UTF-8"> - 设置编码格式
            <title>HTML的标题</title>
       </head>
       <body> - 定义显示在浏览器中的内容
       </body>
       </html>
     * HTML元素
       * 空元素 - 只有开始标签，没有结束标签
       * 起始元素 - 具有开始标签和结束标签
     * HTML属性
       * 通用属性 - 几乎所有的元素都具有
         * id、name、style和class
       * 私有属性 - 某个元素独有的属性
       * 格式 - 属性名="属性值"
   * HTML元素
     * 标题 - <h1> ~ <h6>
       * 在实际开发中，常用的<h1> ~ <h3>
       * 最常用的是<h1>
         * 搜索引擎抓取HTML页面
     * 段落 - <p></p>
       * 特点 - 自动换行，行间距比较大
       * <hr> - 水平线
       * <br> - 换行
     * 列表
       * 有序列表
         <ol>
            <li></li>
         </ol>
       * 无序列表
         <ul>
            <li></li>
         </ul>
       * 定义列表
         <dl>
            <dt></dt>
            <dd></dd>
         </dl>
     * 链接远胜于 - <a>
       <a href="跳转的地址"></a>
       * 默认的作用 - 实现页面之间的跳转
       * 其他的作用
         * 回到顶部 - 锚点(#name)
         * 发送邮件 - href="mailto:"
今天的内容:
 * 图片元素 - <img>
   * 属性
     * src - 设置引入图片的地址
     * alt - 设置图片备选文本内容
     * width - 设置图片显示的宽度
     * height - 设置图片实现的高度
   * 注意 - 同时设置宽度和高度时，注意图片的比例
 * 表格元素
   * 概念 - 表格一定具有行和列
   * <table>元素 - 表示表格
     * <tr>元素 - 表示表格中的行
     * <th>元素 - 表示表格中的表头
     * <td>元素 - 表示表格中的单元格
   * <table>元素的属性
     * border - 设置表格的边框(双边框)
     * width - 设置表格的宽度
   * 表格分为
     * <thead> - 包含<th>表头，显示在表格的最前面
     * <tbody> - 表格中的数据内容
     * <tfoot> - 表格的表尾，显示在表格的最后面
   * 带标题表格
     <caption>表格的标题</caption>
   * 跨行或跨列的表格
     * rowspan - 表示跨行
     * colspan - 表示跨列
   * 页面布局
 * 表单元素 - <form>
   表单内组件元素
   * 输入框 - <input>
 * SVN操作
   * 创建SVN版本库
     1. 在本地磁盘中，创建一个空的文件夹，并且命名(尽量不要使用中文)
     2. 在当前的空文件夹的地址中输入 "cmd"，回车
     3. 在命令提示符窗口中输入
        svnserve -d -r 当前目录的地址
   * 在客户端操作
     1. 在本地磁盘中，创建一个空的文件夹，并且命名
     2. 进入到新创建的文件夹中，鼠标右键【SVN检出】
        版本库URL: SVN服务器的地址
          svn://192.168.10.165
        检出至目录 在本地存储SVN服务器的内容
          C:\Workspace\0705
   * 新增文件
     1. 鼠标右键创建新的文件
     2. 鼠标选中当前文件，右键【SVN】->【加入】
     3. 鼠标选中当前文件，右键【提交】
   * 修改文件
     1. 直接修改目标文件
     2. 鼠标选中当前文件，右键【提交】
   * 删除文件
     1. 删除目标文件
     2. 鼠标选中当前文件，右键【提交】
